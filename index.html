Comment utiliser

Workflow

  1. Configurer le site (LED haut-gauche).
  2. Régler l'instant (LED haut-milieu) — défaut "maintenant".
  3. Ajouter des cibles via + Cible ou Catalogue.
  4. Touche ↻ Calculer pour rafraîchir les éphémérides.
  5. Touche une ligne pour voir les détails et copier les coordonnées.
  6. Dans le Seestar : SkyAtlas → Objects → My Favorites → More → Customize → coller RA et Dec.

Code LED

  • Hauteur ≥ 30° et magnitude OK
  • Hauteur entre seuil min et 30°
  • Trop faible (magnitude > seuil)
  • Sous l'horizon

Installation en tant qu'app iOS

Ouvre cette page dans Safari → bouton Partager → Sur l'écran d'accueil. L'app s'ouvre ensuite en plein écran sans barre Safari.

Hébergement

GitHub Pages, Netlify, ou n'importe quel hébergeur HTTPS. La géolocalisation et le PWA nécessitent HTTPS.

Source des éphémérides

JPL Horizons API (ssd.jpl.nasa.gov). Précision sub-arcseconde pour les astéroïdes numérotés. Coordonnées astrométriques ICRF/J2000 — directement compatibles avec le Seestar.
